## compared to FunkHub, (aka. features)
this project:
 - has mobile support (yay)
 - first-class support for linux (arch linux gng wya?)
 - uses tauri (lighter binaries and memory usage)
 - minimizes to tray for less cpu and memory usage when you start a game
 - doesn't duplicate mods like they never existed
 - has **_themes_**
 - uses the _modern_ gamebanana v11 api (maybe funkub uses it idk i don't heavily look funkhub source)

## installation

### all linux distros
this program is avaliable to all distros via 3 main methods

#### snapcraft
```shell
snap install funkincenter
```

#### flatpak
```shell
flatpak install flathub dev.oofybruh9.funkincenter
```

#### appimage

avaliable on gh releases  
  
nightlies on github actions


### distro-specific releases
nothing special, just releases avaliable in official distro package managers

#### AUR (arch linux)
```shell
yay -S funkincenter
```
OR
```shell
yay -S funkincenter-git
```

#### debian
```shell
sudo apt install funkincenter
```
OR .deb in gh releases  
nightlies on gh actions

#### RPM (redhat-based distros)
